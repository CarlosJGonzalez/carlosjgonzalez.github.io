At Vertilink, I orchestrated the technical strategy for a high-traffic college campus parking system, transitioning the institution from legacy physical hardware to a secure, mobile-first ecosystem.

The Challenge

The goal was to replace costly, physical RFID credentials with a scalable digital system. The architecture needed to ensure real-time validation of user status for thousands of concurrent students and faculty members while maintaining zero-latency access.

Architectural Solution

  • Protocol Design: Defined the encrypted BLE handshake between user mobile devices and hardware for secure, low-latency discovery and connection.
  • Dynamic Authorization: Built the validation logic to restrict access exclusively to authorized and active students via real-time enrollment checks before a pulse was sent.
  • Hardware Integration: Engineered a hardware bridge using an Ethernet relay connected to a modem; the system triggered the physical gate via a network pulse upon successful authentication.
  • Infrastructure Resilience: Centralized the modem-connected relay setup to allow for remote management and high uptime across the campus network.
